核心能力

不只是聊天 — 一个完整的 AI 工作台

openLoom 从底层构建为多智能体协作平台，每个功能都经过精心设计，让你的 AI 工作流更加高效。

多智能体协作

AgentPool 管理多个独立 Agent，每个拥有自己的记忆、工具集和人格。支持子智能体委派，9 状态生命周期精确控制。

知识图谱记忆

PatternAggregator 追踪你的使用模式，自动生成认知摘要。PersonaProvider 构建用户画像，让 AI 越来越懂你。

插件 & Skill 市场

WASM 编译管线支持多格式外部 Skill（Claude Code、OpenClaw）。内置市场一键安装，无限扩展 AI 能力边界。

MCP 协议支持

完整实现 Model Context Protocol，连接外部工具和数据源。LSP 集成提供代码智能，让你的 AI 直接读写你的项目。

定时任务调度

内置 Cron 引擎，支持定时触发 Agent 执行。每日摘要、自动检查、周期性任务 — 设置一次，持续运行。

本地优先 · 隐私至上

所有数据存储在本机，支持自定义数据保留策略。沙箱安全层 + Landlock 隔离，你的数据你做主。

技术深度

为专业用户打造的硬核能力

openLoom 不是简单的 Chat UI 包装器。它是一个完整的 AI 工程平台。

Agent 引擎

9 状态生命周期 (Idle → Thinking → Acting → WaitingForSubagent → Completed)。支持子智能体 Fork、中断恢复、错误隔离。每个 Agent 独立配置模型、工具集、系统提示词。

AgentPool Subagent Streaming

知识系统

向量搜索 + 模式聚合 + 用户画像三层记忆架构。Knowledge Graph 可视化面板 (force-graph-2d)，认知自动提取与持久化，Persona 随时间演化。

Vector Search Persona KG Panel

Skill & 插件系统

内置 5 个 Rust Native Skill + WASM 编译管线。兼容 Claude Code Skill (SKILL.md + scripts/) 和 OpenClaw 格式。Marketplace 一键安装、版本管理、依赖解析。

WASM Marketplace Multi-format

LSP 代码智能

深度集成 Language Server Protocol，AI 可直接理解你的代码结构。支持诊断信息、跳转定义、引用查找、代码补全，让 Agent 成为真正的编程搭档。

LSP Code Editor Diff View
